A:TNG Values

by A:TNG - Mar 10th, 2014 - posted in Mission, Values

By working Agile and helping others do so we have come to value:

  • Consensus protocols over accountable hierarchies
  • Working teams over certified authorities
  • Metric feedback over empowered insight
  • Adaptive patterns over prescriptive frameworks

That is, while there is value in the items on the right, we value the items on the left more. And while we honor the original Manifesto values, we honor these values equally.
